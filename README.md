# projeto-estrutura
No arquivo de teste do 'DoublyLinkedList' temos os metodos:insert,remove,clearFrom,getNext,getPrev,isEmpty,size
O metodo insert :método insert adiciona NÓ dentro da lista.
O metodo remove :Método "remove" remove nó de dentro da lista.
O metodo clearFrom :clearFrom vai remover nó(s) a partir de um nó.
O metodo getNext :vai retornar o próximo nó a partir de um elemento de um nó.
O metodo getPrev :vai retornar o nó anterior partir de um elemento de um nó.
O metodo isEmpty :Retorna se a lista está vazia.
O metodo size :Retorna a quantidade de elementos.


No arquivo de teste do 'CircularLinkedList' temos os metodos:insert,remove,getCurrent,isEmpty,size,clear
O metodo insert :método insert adiciona NÓ dentro da lista.
O metodo remove :Método "remove" remove nó de dentro da lista. OBS:NÃO É POSSIVEL REMOVER UM ELEMENTO QUE NÃO ESTEJA DENTRO DE UM DOS QUATRO CASOS DE REMOÇÃO
O metodo getCurrent :Retorna a tarefa atualmente apontada por currentNode.
O metodo isEmpty :Retorna se a lista for vazia.
O metodo clear :Remove todos os elementos da lista.
O metodo size :Retorna a quantidade de elementos.

solução problema(Lista Circular):
addTask: Utiliza o método insert para adicionar uma nova task a lista.
removeTask: utiliza o método remove para remover uma task da lista.
nextTask: utiliza o método getCurrent para avançar uma tarefa na lista.  
currentTask: este método retorna a tarefa atual da lista.
